
==============================
Local development with MCloud
==============================

MCloud applies it's best effort to provide tooling that help making developer
life easier.

For developer MCloud means:

- No need to install software of every project on his Mcahine
- Repeatable application environment
- No care about development/production differences
- Nice urls like myapp.mcloud.lh
- Simple communication with services: *mysql -h mysql.myapp.mcloud.lh*


This chapter will describe how MCloud operates, some useful patterns for
MCloud development.
